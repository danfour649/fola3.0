# First 90 Days Plan

This plan keeps startup execution focused while staying compliant.

## Days 1-15: Foundation

- Finalize services, target customers, and service area.
- Complete registration decisions and filings (SNB + CRA as needed).
- Identify permit/licence needs using BizPaL and municipality checks.
- Get insurance quotes and bind required policies.
- Build your core templates (quote, agreement, invoice, checklist).

## Days 16-30: System setup

- Set up bank account and bookkeeping workflow.
- Build a simple pricing model:
  - Base service rates
  - Add-ons
  - Travel/parking rules
  - Commercial vs residential pricing
- Define standard operating procedure (SOP) per service type.
- Set up quality checks and client communication scripts.

## Days 31-60: Controlled launch

- Start with a limited number of clients/routes.
- Validate actual labor time vs quoted time.
- Track supply usage per service to refine pricing.
- Collect client feedback and adjust scope language.
- If hiring, complete documented onboarding and safety training.

## Days 61-90: Stabilize and scale

- Review margins by service type and client segment.
- Remove low-margin services unless strategically necessary.
- Standardize renewal/cross-sell processes.
- Build referral pipeline and retention workflow.
- Set recurring compliance calendar for all filings/renewals.

---

## Weekly scorecard (recommended)

Track these weekly:

- New leads
- Quotes sent
- Win rate
- Active clients
- Reclean/callback rate
- Average revenue per job
- Gross margin estimate per job
- Accounts receivable aging
