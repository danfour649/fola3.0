# Compliance Checklist (New Brunswick Cleaning Company)

Use this as a living checklist. Mark each item done only when documented.

## A) Business setup and registration

- [ ] Confirm legal structure (sole proprietorship / partnership / corporation)
- [ ] Select business name
- [ ] Register name/entity through Service New Brunswick (if required)
- [ ] Save proof of registration in repository

## B) CRA and tax setup

- [ ] Obtain Business Number (BN) through CRA Business Registration Online (BRO)
- [ ] Determine if GST/HST registration is required now or voluntary
- [ ] If registered for GST/HST, record effective date and filing frequency
- [ ] If hiring, open payroll deductions account (RP)
- [ ] Create process for tax remittance calendar reminders

## C) Permits and local requirements

- [ ] Run business profile through NB BizPaL
- [ ] Confirm municipal requirements (zoning, home occupation, signage, business licence if applicable)
- [ ] File/obtain any required permits and store confirmation documents

## D) Insurance and liability controls

- [ ] Obtain commercial general liability (CGL) insurance
- [ ] Obtain commercial auto policy (if using vehicles for business)
- [ ] Confirm coverage for tools/equipment
- [ ] Evaluate bonding requirement for commercial contracts
- [ ] Set written procedure for incident reporting and claims support

## E) WorkSafeNB and safety compliance

- [ ] Determine whether mandatory WorkSafeNB coverage applies
- [ ] If mandatory, register within required timeline
- [ ] If not mandatory, assess voluntary/personal coverage options
- [ ] Build onboarding safety orientation checklist
- [ ] Set up WHMIS and chemical handling process
- [ ] Create PPE policy and replacement cadence

## F) Employment standards and HR setup (if hiring)

- [ ] Verify current NB minimum wage and overtime rules
- [ ] Draft compliant offer letter template
- [ ] Draft time tracking and payroll record process
- [ ] Define vacation/public holiday handling
- [ ] Build policy for leaves and termination process

## G) Client and contract controls

- [ ] Standardize estimate/quote template
- [ ] Standardize service agreement terms
- [ ] Add scope boundaries (what is included/excluded)
- [ ] Add cancellation, rescheduling, and late payment terms
- [ ] Set process for pre/post-service quality check

## H) Finance and operations

- [ ] Open dedicated business bank account
- [ ] Set bookkeeping process and chart of accounts
- [ ] Track job-level profitability
- [ ] Implement invoice numbering and payment tracking
- [ ] Set monthly close and KPI review checklist

## I) Annual and recurring obligations

- [ ] Annual return/renewal reminders (as applicable to structure)
- [ ] Insurance renewal reminders
- [ ] GST/HST filing reminders (if registered)
- [ ] Payroll remittance and year-end slip reminders (if hiring)
- [ ] WorkSafeNB payroll reporting reminders (if applicable)
